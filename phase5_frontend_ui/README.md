# Phase 5: Frontend UI

![Build Status](https://img.shields.io/badge/Build-Passing-brightgreen)
![Status](https://img.shields.io/badge/Status-Green-brightgreen)

This directory houses the React / Vite dashboard visualizer displaying current pipeline run logs, graphs, charts, and product reviews.

## Contents
- `src/App.jsx`: Responsive layout dashboard with analytics widgets and cards.
- `src/index.css`: Glassmorphic styling rules, CSS variables, and layout systems.
- `index.html`: Entrypoint template for React rendering.

## Setup & Running
1. Install Node modules:
   ```bash
   npm install
   ```
2. Run development build server:
   ```bash
   npm run dev
   ```
3. Compile production asset build:
   ```bash
   npm run build
   ```
